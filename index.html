TopSBM: Topic Models based on Stochastic Block Models

Topic models have wide applicability, but its most popular flavours (based on Dirichlet priors, such as LDA) make unreasonable asumptions about the data which severly limit its applicability. Here we explore an alternative way of doing topic models, based on stochastic block models (SBM).

Example: 63 Wikipedia articles related to Physics

Install

An example of how to use TopSBM is shown here. To run it yourself, you need to:

  1. Install graph-tool :
    • see detailed Installation Instructions .
    • or using Docker :
      docker pull tiagopeixoto/graph-tool
    • or using Conda (only for linux).
  2. Download the package SBM topic models:
    • Using git :
      git clone https://github.com/martingerlach/hSBM_Topicmodel.git
    • or: download zip file.
  3. Explore the tutorial using Jupyter notebooks:
    • jupyter-notebook TopSBM-tutorial.ipynb

References:

  • M. Gerlach, T. P. Peixoto, and E. G. Altmann, A network approach to topic models, [arXiv:1708.01677] .
